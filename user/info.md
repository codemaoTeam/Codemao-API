# 个人信息

> 获取个人信息  
> `/api/user/info` `GET` `需要cookies`

## 返回内容

|     key     |             value              |  type  |
| :---------: | :----------------------------: | :----: |
|     age     |              年龄              |  int   |
|   avatar    |            头像链接            | string |
| discription |              简介              | string |
|    doing    |           正在做的事           | string |
|    email    |              邮箱              | string |
|    gold     |              金币              |  int   |
|     id      |            用户 id             |  int   |
|     sex     |      性别(m 为男，s 为女)      | string |
|    level    | 等级（1 为潜力新星，以此类推）  |  int  |
|     qq      |             qq 号              |  int  |
